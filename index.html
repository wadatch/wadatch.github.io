Browser OCR Cropper

写真・画像・PDF から文字を抽出できる無料 OCR ツール。ファイルをサーバーに送らず、すべてブラウザ内で処理するので安心して使えます。

  • 日本語・縦書きに対応
  • 画像のトリミングをしてから OCR 可能
  • インストール不要・完全無料

WordPress Plugins

WordPress Plugin

WP Entry Form

ボランティア募集などの申し込みフォームを WordPress に設置できるプラグイン。管理画面から項目を自由に定義でき、コーディング不要でフォームを作れます。

  • ショートコード / Gutenberg ブロックで埋め込み
  • 確認画面・ファイル添付・同意チェックに対応
  • 応募一覧の検索・CSV エクスポート
  • ハニーポット & レート制限によるスパム対策

WP Google Login

WordPress のログイン画面に「Google でログイン」を追加するプラグイン。Google One Tap にも対応し、メールアドレスの照合や新規アカウントの自動作成ができます。

  • OAuth / Google One Tap 対応
  • メールドメインに応じたロール自動割り当て
  • ログイン後処理を拡張できるフック付き

Kintone Plugin

Kintone PDF Viewer Plugin

kintone の添付ファイル（PDF・テキスト）をダウンロードせずにその場で閲覧できるプラグイン。ファイルリンクをクリックするとモーダルで表示されます。

  • PDF のズーム・ページ移動・回転に対応
  • テキストファイルも等幅フォントで表示
  • そのままダウンロードも可能
